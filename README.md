# 🏔️ Bootcamp Avalanche — Guía Oficial

> **CriptoUNAM** · Infraestructura Web3 con finalidad instantánea y escalabilidad infinita.

Esta es la guía de referencia del **Bootcamp Avalanche**: 4 semanas, 8 sesiones presenciales para dominar Avalanche, construir un MVP funcional y aprender a lanzar tu propia L1.

---

## 🛠️ ¿Qué lograrás?

| Área | Contenido |
|------|-----------|
| **Infraestructura** | Red Primaria (P, X y C-Chain), modelo Avalanche9000 |
| **Smart Contracts** | Solidity optimizado para Avalanche, Foundry/Hardhat |
| **Soberanía Digital** | Configurar, lanzar y gestionar tu propia L1 personalizada |
| **Interoperabilidad** | Teleporter y Avalanche Warp Messaging (AWM) |
| **Business & Launch** | Modelos de negocio Web3 y presentación ante panel de expertos |
