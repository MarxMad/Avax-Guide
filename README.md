# 🏔️ Bootcamp Avalanche — Guía Oficial

> **CriptoUNAM** · Infraestructura Web3 con finalidad instantánea y escalabilidad infinita.

Esta es la guía de referencia del **Bootcamp Avalanche**: 4 semanas, 8 sesiones presenciales para dominar Avalanche, construir un MVP funcional y aprender a lanzar tu propia L1.

---

## 🚀 Cómo usar esta guía

- **Primera vez:** empieza por [Semana 1 - Sesión 1](./semana-1/01-fundamentos-core-wallet.md) (fundamentos y Core Wallet).
- **Cada sesión** incluye objetivos, contenido, diagramas, checklists y enlaces a documentación oficial.
- **Recursos centrales:** [enlaces](./recursos/enlaces.md) y [herramientas](./recursos/herramientas.md) en la carpeta `recursos/`.

---

## 🛠️ ¿Qué lograrás?

| Área | Contenido |
|------|-----------|
| **Infraestructura** | Red Primaria (P, X y C-Chain), modelo Avalanche9000 |
| **Smart Contracts** | Solidity optimizado para Avalanche, Foundry/Hardhat |
| **Soberanía Digital** | Configurar, lanzar y gestionar tu propia L1 personalizada |
| **Interoperabilidad** | Teleporter y Avalanche Warp Messaging (AWM) |
| **Business & Launch** | Modelos de negocio Web3 y presentación ante panel de expertos |

## 📌 Detalles del evento

- **Fechas:** Lunes y Miércoles (2–25 de marzo)
- **Horario:** 17:00–19:00 hrs
- **Sede:** Anexo Facultad de Ingeniería, Aula Cisco UNAM
- **Costo:** Gratuito (cupo limitado)
- **Certificación:** Certificados on-chain (POAPs/NFTs)

**Requisitos:** Laptop propia y conocimientos básicos de programación (preferiblemente JavaScript/Solidity). Para la Sesión 1 necesitas instalar [Core Wallet](https://core.app/); para la Sesión 2, Node.js (v18+) y Foundry o Hardhat (ver [herramientas](./recursos/herramientas.md)).
