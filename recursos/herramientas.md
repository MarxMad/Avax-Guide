# Herramientas recomendadas

Lista de software y servicios útiles para el bootcamp.

---

## Wallets

- **Core Wallet** — Extensión para navegador; soporte nativo Avalanche (P, X, C-Chain).
- **MetaMask** — Compatible con C-Chain; agregar red Fuji/Mainnet manualmente.

---

## Desarrollo de contratos

- **Foundry** — Compilación, tests, fuzzing, despliegue (`forge`, `cast`, `anvil`).
- **Hardhat** — Entorno JavaScript/TypeScript para compilar, testear y desplegar.
- **Remix** — IDE en navegador; conectar a Fuji vía “Injected Provider”.

---

## Frontend y librerías

- **Ethers.js (v6)** — Conexión a RPC, contratos, wallets.
- **Viem** — Alternativa moderna, TypeScript, compatible con Avalanche (chains predefinidas).
- **React + Vite** — Setup rápido para dApps.
- **Wagmi + RainbowKit** — Hooks y UI de conexión de wallet (opcional).

---

## Redes locales y nodos

- **Anvil** (Foundry) — EVM local para pruebas.
- **Avalanche CLI** — Crear y gestionar tu L1 (subnet) local.
- **Subnet-EVM** — VM para tu L1 EVM.

---

## Seguridad y análisis

- **Slither** — Análisis estático de contratos Solidity.
- **Aderyn** — Análisis estático (alternativa).
- **Foundry fuzz tests** — Pruebas aleatorias de invariantes.

---

## Indexación y datos

- **Snowtrace** — Explorador y API para C-Chain (Fuji y Mainnet).
- **The Graph** — Subgraphs para indexar eventos en Avalanche.

---

## Otros

- **Node.js** — v18 o superior (LTS).
- **Git** — Control de versiones.
- **Editor** — VS Code / Cursor con extensiones Solidity y ESLint.
