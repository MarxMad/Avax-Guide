# Imágenes Semana 1 — Builders Hub

Las guías de la Semana 1 dejan espacio para imágenes de referencia. Puedes usar capturas o diagramas del **Avalanche Builders Hub**:

- **Docs:** https://build.avax.network/docs  
- **Arquitectura AvalancheGo:** https://build.avax.network/docs/nodes/architecture  
- **PlatformVM (P-Chain):** https://build.avax.network/docs/primary-network/platformvm-architecture  

### Cómo usar

1. Descarga o captura la imagen desde el Builders Hub.
2. Guárdala en esta carpeta (`semana-1/assets/`) con el nombre indicado en cada guía.
3. En los `.md` ya está referenciada con `./assets/nombre.png`.

### Nombres sugeridos (opcionales)

| Archivo | Uso en guía |
|---------|-------------|
| `primary-network-architecture.png` | Sesión 1 — Red Primaria |
| `consensus-avalanche.png` | Sesión 1 — Consenso |
| `c-chain-ecosystem.png` | Sesión 2 — C-Chain |
| `developer-workflow.png` | Sesión 2 — Flujo de desarrollo |

Si no añades estos archivos, las guías siguen siendo útiles: los diagramas Mermaid y el texto ya explican el contenido.
